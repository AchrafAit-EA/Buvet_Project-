📌 Description du projet

Site web statique pour une buvette nommée **"Maison du Savoir"**, développé avec **HTML, CSS et JavaScript**. Ce projet présente une interface moderne et simple permettant aux utilisateurs de découvrir les produits proposés par la buvette.

Le site inclut une présentation générale de la buvette, un menu détaillé avec des images ainsi qu'une section de contact pour localiser l'établissement.

🚀 Fonctionnalités

* Affichage du menu de la buvette 🍔
* Présentation des plats avec images 📸
* Description de la buvette 🏪
* Section de contact avec localisation 📍
* Interface simple et responsive

🛠️ Technologies utilisées

* HTML
* CSS
* JavaScript

🎯 Objectif

L'objectif de ce projet est de créer un site web statique simple et attractif pour présenter une buvette, en mettant en avant les produits, l'expérience utilisateur et le design.
